# Matrix Mania

A menu-driven C program implementing basic matrix operations
using dynamic memory allocation.

## Features

- Matrix Addition
- Matrix Subtraction
- Matrix Multiplication
- Matrix Transpose

## Concepts Used

- Dynamic 2D arrays
- Pointer-based memory management
- Modular function design
- Input validation

## Compile

gcc matrix_mania.c -o matrix_mania

## Run

./matrix_mania
