# C-Basics

A collection of C programming projects, examples, and practice problems built while learning the language. This repository covers fundamental concepts ranging from basic programming exercises to file handling and dynamic memory management.

## Projects

### 📐 Patterns

A collection of pattern-printing programs ranging from basic index-based patterns to intermediate geometric designs. These programs help strengthen understanding of loops, nested loops, and logical thinking.

### 🧮 Matrix Mania

A menu-driven matrix calculator implemented using dynamic memory allocation.

**Features:**

* Matrix addition
* Matrix subtraction
* Matrix multiplication
* Matrix transpose
* Dynamic memory allocation
* Menu-driven interface

### 📒 CLI Contact Book

A command-line contact management system that stores data using binary files and an indexing mechanism.

**Features:**

* Add contacts
* List contacts
* Search contacts by name
* Delete contacts (logical deletion)
* Binary file storage
* Index-based searching using file offsets
* Automatic database file creation on first execution
