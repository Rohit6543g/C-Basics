# C-Basics

An organized library of examples and problems covering all the basic concepts of C programming.

## Contents

- Patterns – Index-based and geometric pattern problems
- Matrix Mania – Menu-driven matrix operations using dynamic memory

## Patterns

Many different types of pattern problems added to the repo which includes the basic index based pattern to intermediate level geometric pattern problems.

## Matrix Mania

A menu-driven C program implementing basic matrix operations
using dynamic memory allocation.
